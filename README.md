# 🚴‍♂️ Cycle - Modern Bike Shop Web App

A clean, premium, and dynamic landing page for a fictional bike shop, built using **HTML, CSS (Dark Mode), and JavaScript**. This project is designed to showcase a sleek interface with glassmorphism and micro-animations that promotes bicycle sales and services.

## ✨ Features

- **Dark Mode Aesthetic:** A sleek neon-cyan and deep blue color palette with glassmorphism effects.
- **Dynamic Interactions:** Hover effects, smooth transitions, and floating animations on elements.
- **JavaScript Logic:**
  - Dynamic Glass Header on scroll.
  - Interactive image carousel in the Hero section.
  - Client-side form validation for the Contact Us section.
  - Mobile-responsive navigation toggle.
- **Multi-Page Layout:** Separate pages including Home, About, Our Cycle, News, and Contact.
- **Fully Responsive:** Optimized for mobile, tablet, and desktop viewing.

## 🚀 Getting Started

To run this project locally:

1. **Clone the repository:**
   ```bash
   git clone https://github.com/basem-samir/cycle.git
   ```
2. **Navigate to the project directory:**
   ```bash
   cd cycle
   ```
3. **Open the project:**
   Simply double-click the `index.html` file or open it directly in your web browser.
